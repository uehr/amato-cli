What's amATO

    amATO / あまと

    プレゼンテーションツール

    Markdownで書いたプレゼン資料をお手軽にリッチなスライド資料へ

    お手軽にスライドデザインを切り替えられる

Demo

このスライド自身

Installation

npmパッケージとして公開

$ npm install amato-cli

Usage

    Markdownで資料作成

sample/sample.md# This is sample
## slide1 :)
---
## slide2 :X
---
## slide3 :D

    MarkdownファイルをHTMlへ変換

$ amato-cli sample/sample.md -o sample
Generating slides.
■ Source: sample/sample.md
■ Output: sample/sample.html
■ Style: default
Complate.

    生成されたhtmlファイルをブラウザで開く

    $ open sample.html
